# ASD Hardneing Guides

-------------------

This repo is a bunch of stuff to implement hardening on Win11/Win10 devices based of ASD guideance.
Mainly stuff that ASD says to do, but doesn't give templates to implement

There is GPO exports of the hardening guides for Win10/11 and Office 365. <br>
Microsoft provides settings catalog imports for Intune managed devices, but there isn't anything for on-prem things.

Based of the 16-Jul-24 version of the [Windows 10/11 Hardening Guide](https://www.cyber.gov.au/business-government/protecting-devices-systems/hardening-systems-applications/system-hardening/hardening-microsoft-windows-11-workstations) <br>
and 24-Jul-23 version of [Office 365, 2021, 2019 and 2016 Hardening Guide](https://www.cyber.gov.au/business-government/protecting-devices-systems/hardening-systems-applications/system-hardening/hardening-microsoft-365-office-2021-office-2019-and-office-2016) <br>

There is also versions of the scripts found [here](https://blueprint.asd.gov.au/configuration/intune/devices/scripts/), now with detection logic as well

Blah blah blah no warranties, do testing, etc etc <br>
I did this so you hopefully didn't have to.
